# 🌟 Clarinha Bot - WebSite

Este é o repositório do site oficial da **Clarinha**, um bot focado em economia, diversão e apostas para o Discord. O site permite que os usuários realizem o login via OAuth2 do Discord para resgatar recompensas diárias (Daily) e consultar comandos.

## 🚀 Funcionalidades

*   **Sistema de Daily:** Resgate de moedas (Candys) diárias com verificação de Captcha.
*   **Autenticação Discord:** Login seguro utilizando OAuth2 para identificar usuários.
*   **Painel de Comandos:** Lista completa de funcionalidades do bot.
*   **Página Premium:** Informações sobre planos VIP e benefícios.
*   **Interface Responsiva:** Design limpo e moderno adaptado para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas

*   **Front-end:** HTML5, CSS3, JavaScript.
*   **Back-end:** Node.js com Express.
*   **Segurança:** Google reCAPTCHA v2 e variáveis de ambiente para proteção de dados sensíveis.
*   **APIs:** Integração direta com a API do Discord.

## ⚙️ Como rodar o projeto

1.  Clone o repositório:
    ```bash
    https://github.com/fssamuelbs/ClarinhaWebSite/
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Configure o arquivo `.env` com suas credenciais:
    *   `CLIENT_ID`
    *   `CLIENT_SECRET`
    *   `CAPTCHA_SECRET`
    *   `WEBHOOK_AUTH_URL`
4.  Inicie o servidor:
    ```bash
    npm start
    ```
